      编程日记

      在自己鸽了两周的Web作业后，今天的我从期末前的各种作业中爬出来，学习了一下JS语言.

      JS基本语法

      在学过Python后看JS感觉还OK，两类面向对象的语言很相似，所以基本语法没有多练习了，在以后的实践中慢慢磨合吧

      1.准备工作

      • 在VS里更新了Nodejs的组件(但是感觉用起来不是很舒服，VS都会帮你打包成一个文件夹，在命令行运行的时候有点不爽)

      • VS Code 基本编的JS代码都是用的VS Code编的，很方便，但是里面Debug的配置搞来搞去好麻烦

      • Chrome 浏览器里的Console运代码看得舒服，还很流畅，所以就把代码粘贴在这里Debug了

      2.学习内容

      • 首先把JS的基本语法看了一遍(廖雪峰网站上的)

      • 具体了解了一下模块概念，以及模块之间的引用功能

        • module.exports = {}(经常使用)

        • exports.name = funsname

        • var 变量名 = require(‘文件路径’)[‘引用的对象’];

      • 更深入了解了同步异步概念

      • 学习了文件读取操作，以及文件写入操作

      • Pipe概念：将文件分成小部分，分别进行读取以及写入，不至于把电脑内存占满

      • 学习了解了HTTP：

        • 浏览器向服务器发送HTTP请求

          • GET还是POST

          • 域名

          • …

        • 服务器向浏览器返回HTTP响应

          • 响应代码

            • 200成功

            • 3xx重定向

            • 4xx用户请求错误

            • 5xx服务器错误

          • 响应类型

          • …

        • 浏览器再次请求其他资源，重复前两步

      • HTTP 格式

        • POST 请求格式:

          POST /path HTTP/1.1
          Header1: Value1
          Header2: Value2
          Header3: Value3

          body data goes here…

        • HTTP 响应格式

          200 OK
          Header1: Value1
          Header2: Value2
          Header3: Value3

          body data goes here…
